# Ethiopian Entrance Exam Calculator (Python)

This Python program calculates the Ethiopian entrance exam total score from six subjects and provides a promotion, failure, or remedial decision based on predefined rules. It handles gender-based remedial eligibility and demonstrates basic Python concepts such as input handling, conditional logic, and arithmetic operations.

## Subjects
The program considers scores from the following subjects:
- English
- Mathematics
- Aptitude
- Physics
- Chemistry
- Biology

## Features
- Calculates the total score of the exam
- Provides a promotion, failure, or remedial comment based on the result
- Handles gender-based remedial eligibility
- Beginner-friendly Python code suitable for learning conditionals and input handling

## How to Run
1. Clone or download the repository.
2. Open the `Ethiopian entrance exam calculator.py` file in your Python environment (Pydroid 3, IDLE, VS Code, etc.).
3. Run the script and follow the prompts to enter scores and gender.
4. The program will display the total score and the corresponding comment.

python result_calculator.py
